技术网站与书籍推荐

Posted on 2018-04-20 | | Views:   |   317 words

技术社区

分享一些我常会访问的技术社区

  1. 掘金,很活跃的技术社区

  2. 博客园, 不少大牛

  3. segmentfault.com,类似startoverflow的问答社区，也有不少专栏文章
  4. cnode，关于nodejs的
  5. medium，不少外文优质技术文章

工具网站

  1. mdn
  2. caniuse

书籍

看完入门类的书籍应该对js有些基础的了解，可以上手干活了。

入门类
  1. js DOM编程艺术
  2. js 高级程序设计
  3. js 编程艺术
  4. effective js

前端很容易碰到瓶颈和迷茫，看下下面的书可能有帮助。

提升类
  1. HTTP权威指南，还不了解网站从url到网页的过程吗？
  2. webkit技术内幕，不了解一下浏览器？
  3. 深入浅出nodejs，几年前的书，仍然经典
  4. 编译器设计，造轮子必看
  5. 你不了解的js，深入了解js
  6. 算法，这年头不懂快排不好找工作…

最后
